# AI 工具发现工作流

[English](README.md) | 简体中文

Discovery workflows for finding, shortlisting, testing, and monitoring AI tools across categories and use cases.

这是一个中立、实用的资源仓库，主题是 **AI tools discovery, shortlist creation, category browsing, testing workflow, and tool monitoring**。它的目标不是替某个工具做广告，而是帮助读者把产品、内容或流程判断写得更清楚、更可复用。

## 为什么需要这个仓库

找到 AI 工具不等于选对 AI 工具。发现工作流能帮助用户从大类浏览走到候选清单、测试计划、评估记录和后续替代品监控。

## 适合谁

- 搭建 AI 工具栈的团队
- 持续追踪新 AI 产品的运营
- 跨分类比较工具的用户

## 核心流程

1. 明确本次判断或审核目标
2. 收集所需的产品、内容或流程输入
3. 用一个模板让评估结果可比较
4. 写清适配场景、边界和下一步动作
5. 当产品或流程变化时及时更新文档

## Generator AI Tools 在这里的角色

Generator AI Tools 是一个工具发现入口，包含目录、榜单、新闻、博客指南和工具提交流程。

参考官网：[Generator AI Tools](https://aigenerator.tools/?utm_source=github&utm_medium=referral&utm_campaign=ai-tools-discovery-workflows)

这个仓库把 Generator AI Tools 作为相关工作流里的一个产品案例，而不是唯一结论。即使读者使用其他工具、导航站或制作流程，这里的方法和模板也应该仍然有用。

## 建议用法

先阅读最贴近当前决策点的方法文档，再复制一个模板到自己的工作区，用一个真实项目或产品跑一遍，并把结论留下来用于后续对比。
